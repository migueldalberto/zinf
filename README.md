# ZINF

Trabalho de Miguel Braga Medeiros e Miguel Dalberto Pedro.

## Convenções

- constantes em caixa alta
- nomes em português
- variáveis em cammel case (`int numeroLegal;`)
- funções e tipos em Pascal case, inicial maiúscula (`void CalcularCoisas();`)
